## 📌 Overview

**Skills International - Student Learning Management System** is a desktop application built using **C# Windows Forms** and **SQL Server**.  
It is designed to simplify the student registration process and manage student information in an efficient and user-friendly way.

This project includes:
- Secure login screen
- Student registration form
- Create, update, delete, and clear operations
- Database integration
- Clean and responsive UI

---

## ✨ Features

| Feature | Description |
|--------|-------------|
| 🔐 Login System | Secure login interface for authorized users |
| 🧑‍🎓 Student Registration | Add new student details with a structured form |
| ✏️ Update Records | Modify existing student information |
| 🗑️ Delete Records | Remove student records from the database |
| 🔄 Clear Form | Reset all input fields quickly |
| 🗃️ Database Integration | Connected to SQL database for persistent storage |
| 🎨 User-Friendly UI | Simple and clean design for easy use |
